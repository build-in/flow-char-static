# 这个是使用antV X6做的一个流程编排工具
##### 1、可以定制化卡片拖拽；
##### 2、可以编辑卡片数据；
##### 3、绘画的数据卡片数据和连接线支持校验；
##### 4、可以查看生成的图表；
##### 5、可以切换不同的历史版本；
##### 6、可以根据对应的表数据生成echarts图；
##### 7、连接线支持实线、虚线，以及各种颜色配置；
